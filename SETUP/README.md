# SETUP — Всё для запуска Crazy Racer

В этой папке всё необходимое для пошагового запуска проекта.

## Файлы

| Файл | Описание |
|------|----------|
| **ШАГ_ЗА_ШАГОМ.md** | Главная инструкция — читай её первой |
| **contracts/CrazyRacerCars.sol** | Код контракта для копирования в Remix |
| **env.example.txt** | Пример переменных окружения |
| **REMIT_DEPLOY.md** | Подробный деплой через Remix |
| **FOUNDRY_DEPLOY.md** | Деплой через Foundry (для опытных) |

## Порядок действий

1. Открой **ШАГ_ЗА_ШАГОМ.md**
2. Выполняй шаги по порядку
3. При деплое контракта используй **REMIT_DEPLOY.md**

Всё.
